## **DISCLAIMER**
-The project was developed during the Los Altos Hacks VIII Hackathon
-The application features a local host website for improved user experience

## **Agent Tools**
- SMS
- Email
- Describe Images
- Retrieve from Notepad (using RAG)
- Push text into notepad

## **Tech Stack**
- LlamaIndex
- Open-CV
- Groq
- GPT-4
- Flask
